# Responsive Web Development

Exercícios práticos de desenvolvimento web responsivo — Univali.

## Estrutura do Projeto

```
responsive_web_development/
├── index.html                  → Menu de navegação
├── aula01-lobo/                → HTML básico: imagem, texto, link
├── aula02-float/               → Posicionamento com float e clear
├── aula03-seletores-css/       → Seletores CSS: classes, IDs, pseudo-classes
├── forum-tematico/             → Fórum Temático: Carta do Leitor
├── unidade-aprendizagem/       → Unidade de Aprendizagem: Estados e Cidades
└── calculadora/                → Calculadora Espacial: JS, CSS e HTML
```

## Páginas

### Menu Principal
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/)

### Aula 01 — Lobo
Introdução ao HTML: estrutura básica, imagem, parágrafo e link externo.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/aula01-lobo/)

### Aula 02 — Posicionamento Float
Técnica de layout com `float: right` e `clear: both`. Uso de `box-sizing: border-box`.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/aula02-float/)

### Aula 03 — Seletores CSS
Uso de seletores de tag, classe (`.meu-btn-laranja`) e ID (`#meu-id`).
Estilização de botões com hover e especificidade CSS.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/aula03-seletores-css/)

### Fórum Temático — Carta do Leitor
Layout editorial responsivo com gradientes e texturas CSS.
Uso de `ul` e `li` para estruturação semântica de uma carta.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/forum-tematico/)

### Unidade de Aprendizagem — Estados e Cidades
Select dinâmico com carregamento via AJAX (XMLHttpRequest).
Backend em PHP retornando JSON com cidades por estado.

### Calculadora Espacial
Calculadora com tema espacial: estrelas animadas, planeta, astronauta flutuante, sons e efeitos neon.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/calculadora/)
