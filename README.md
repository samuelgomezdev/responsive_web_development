# lobo_predador_topo
O lobo é um predador de topo essencial para o equilíbrio dos ecossistemas terrestres. Sua presença regula populações de herbívoros e mantém a estabilidade ambiental. Este relatório analisa seu impacto ecológico, consequências de sua ausência, exemplos reais e regiões com maior risco de desaparecimento.
