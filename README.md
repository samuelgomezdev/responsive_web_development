# Responsive Web Development

Exercícios práticos de desenvolvimento web responsivo.

## Estrutura do Projeto

```
responsive_web_development/
├── aula01-lobo/                  → HTML básico: imagem, texto, link
├── aula02-seletores-css/         → Seletores CSS: classes, IDs, pseudo-classes
├── aula03-float/                 → Posicionamento com float e clear
├── aula04-carta-leitor/          → Layout editorial com CSS avançado
└── aula05-estados-cidades/       → Formulário dinâmico com AJAX + PHP
```

## Aulas

### Aula 01 — Lobo
Introdução ao HTML: estrutura básica, imagem, parágrafo e link externo.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/aula01-lobo/)

### Aula 02 — Seletores CSS
Uso de seletores de tag, classe (`.meu-btn-laranja`) e ID (`#meu-id`).
Estilização de botões com hover e especificidade CSS.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/aula02-seletores-css/)

### Aula 03 — Posicionamento Float
Técnica de layout com `float: right` e `clear: both`.
Uso de `box-sizing: border-box`.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/aula03-float/)

### Aula 04 — Carta do Leitor
Layout editorial responsivo. Background com múltiplos gradientes e texturas CSS.
Uso de `ul` e `li` para estruturação semântica de uma carta.
[Ver página](https://samuelgomezdev.github.io/responsive_web_development/aula04-carta-leitor/)

### Aula 05 — Estados e Cidades
Select dinâmico com carregamento via AJAX (XMLHttpRequest).
Backend em PHP retornando JSON com cidades por estado.
